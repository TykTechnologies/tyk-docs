## Closed Source

The following Tyk components, created and maintained by the Tyk Team, are proprietary and closed-source:

* [Tyk Dashboard](/tyk-dashboard)
* [Tyk Developer Portal](/tyk-developer-portal)
* [Tyk Multi Data Center Bridge](/api-management/mdcb#managing-geographically-distributed-gateways-to-minimize-latency-and-protect-data-sovereignty)
* [Universal Data Graph](/api-management/data-graph#overview)
* [Tyk Operator](/api-management/automations/operator#what-is-tyk-operator)
* [Tyk Sync](/api-management/automations/sync)

If you plan to deploy and use the above components On-premise, license keys are required.
