### List All Key Requests

| **Property** | **Description**          |
| ------------ | ------------------------ |
| Resource URL | `/api/portal/requests`   |
| Method       | GET                      |
| Type         | None                     |
| Body         | None                     |
| Param        | None                     |

#### Sample Request

```http
GET /api/portal/requests HTTP/1.1
Host: localhost
authorization:7a7b140f-2480-4d5a-4e78-24049e3ba7f8
```
