Every field below is present on the traffic log that Tyk Gateway writes to Redis and Tyk Pump distributes to its configured destinations. See [Tyk Pump](/api-management/tyk-pump) for how traffic logs are collected and routed.

## Request

### Method

Request method.

**Example:** `GET`, `POST`.

### Host

Request `Host` header.

**Remarks:** Includes the host and optional port number of the server the request was sent to.

**Example:** `tyk.io`, or `tyk.io:8080` if a port is included.

### Path

Request path, in decoded form.

**Example:** `/foo/bar` for `/foo%2Fbar` or `/foo/bar`.

### RawPath

Request path, decoded but otherwise unchanged from the original request.

**Example:** `/foo/bar` for `/foo%2Fbar` or `/foo/bar`.

### OriginalPath

The client's request path, captured before Tyk Gateway applies any path-modifying middleware, such as stripping the API's listen path. Equal to `Path` unless something changes the path during processing.

**Example:** `/api/v1/users`, when `Path` is `/users` because the API's listen path `/api/v1/` was stripped.

### ListenPath

The API's configured listen path.

**Example:** `/api/v1/`.

### ContentLength

Request `Content-Length` header: the number of bytes in the request body.

**Example:** `10` for a request body of `0123456789`.

### UserAgent

Request `User-Agent` header.

**Example:** `curl/7.86.0`.

### RawRequest

Base64-encoded copy of the request sent from Tyk Gateway to the upstream server.

**Remarks:** Empty unless [detailed recording](/api-management/dashboard-analytics#detailed-recording) is enabled (automatic for GraphQL APIs). A pump can remove the base64 encoding via [`raw_request_decoded`](/api-management/tyk-pump#decode-raw-request-and-raw-response), or omit this field entirely via [`omit_detailed_recording`](/api-management/tyk-pump#omit-detailed-recording).

**Example:** `R0VUIC9nZXQgSFRUUC8xLjEKSG9zdDogdHlrLmlv`.

## Response

### ResponseCode

The integer response code, generated by either Tyk Gateway or the upstream server depending on how the request was handled.

**Example:** `200` for `200 OK`.

### RawResponse

Base64-encoded copy of the response sent from Tyk Gateway to the client.

**Remarks:** Empty unless [detailed recording](/api-management/dashboard-analytics#detailed-recording) is enabled (automatic for GraphQL APIs). A pump can remove the base64 encoding via [`raw_response_decoded`](/api-management/tyk-pump#decode-raw-request-and-raw-response), or omit this field entirely via [`omit_detailed_recording`](/api-management/tyk-pump#omit-detailed-recording).

**Example:** `SFRUUC8xLjEgMjAwIE9LCkNvbnRlbnQtTGVuZ3RoOiAxOQpEYXRlOiBXZWQsIDE2IE5vdiAyMDIyIDA2OjIxOjE2IEdNVApTZXJ2ZXI6IGd1bmljb3JuLzE5LjkuMAoKewogICJmb28iOiAiYmFyIgp9Cg==`.

### RequestTime

Duration of the upstream round trip, equal to `Latency.Total`.

**Example:** `3` for a 3ms round trip.

### Latency

Contains two fields: `upstream`, the round trip between Tyk Gateway sending the request and receiving a response, and `total`, `upstream` plus additional Gateway-side processing such as writing analytics.

**Example:** `{"total":3,"upstream":3}`.

<Note>
This is the round trip time measured from Tyk Gateway's reverse proxy: the sum of leaving Tyk, reaching the upstream, and the response returning to Tyk.
</Note>

## API and Authentication

### APIID

The ID of the requested API definition.

**Example:** `727dad853a8a45f64ab981154d1ffdad`.

### APIName

The name of the requested API definition.

**Example:** `Foo API`.

### APIVersion

The version of the requested API definition, or `Not Versioned` if the API is unversioned.

**Example:** `1`, `b`, or `Not Versioned`.

### OrgID

The Organisation ID of the requested API definition.

**Example:** `5e9d9544a1dcd60001d0ed20`.

### APIKey

The authentication key provided in the request. If no key was provided, Tyk Gateway substitutes a default value.

**Example:** an unhashed `auth_key`, a hashed value such as `6129dc1e8b64c6b4`, or `00000000` if no authentication was provided.

### OauthID

The OAuth client ID, or an empty string if OAuth was not used.

**Example:** `my-oauth-client-id`.

### Alias

The alias of the authenticated identity, blank if unset or the request is unauthenticated.

**Example:** `my-key-alias`.

## Timestamps

### TimeStamp

Generated by Tyk Gateway when it finishes handling the request and writes the traffic log, not when it first received the request. For most requests the difference is small, roughly the request's own `RequestTime`, but it's after the upstream round trip, not before it.

**Example:** `2022-11-16T03:01:54.648+00:00`.

### Day, Month, Year, Hour

Derived from the `TimeStamp` field.

**Example:** for `2022-11-16T03:01:54Z`: Day `16`, Month `11`, Year `2022`, Hour `3`.

### ExpireAt

A future expiry date, used to implement [automated data expiry](/api-management/dashboard-analytics/analytics-storage-management#ttl-indexes) where the storage backend supports it.

**Example:** `2022-11-23T07:26:25.762+00:00`.

## Client

### IPAddress

Client IP address, taken from the `X-Real-IP` or `X-Forwarded-For` header if set, otherwise determined by Tyk Gateway from the request.

**Example:** `172.18.0.1`.

### Geo

Client geolocation, calculated from the IP address using the MaxMind database.

**Example:** `{"country":{"isocode":"SG"},"city":{"geonameid":0,"names":{}},"location":{"latitude":0,"longitude":0,"timezone":""}}`.

### Network

Network statistics. Not currently used.

## Metadata

### Tags

Session context tags, which can refer to the Gateway, API key, Organisation, API definition, and other dimensions.

**Example:** `["key-00000000","org-5e9d9544a1dcd60001d0ed20","api-accbdd1b89e84ec97f4f16d4e3197d5c"]`.

### TrackPath

`true` if the requested endpoint is configured to be tracked, otherwise `false`.

## GraphQL Fields

Only populated for requests to a GraphQL API; absent or empty for other requests.

### ApiSchema

Base64-encoded copy of the API's GraphQL schema.

**Example:** `dHlwZSBRdWVyeSB7IGhlbGxvOiBTdHJpbmd9`.

### GraphQLStats

Details about the GraphQL operation: `Variables`, `RootFields`, `Types`, `Errors`, `OperationType` (`0` Unknown, `1` Query, `2` Mutation, `3` Subscription), `HasErrors`, and `IsGraphQL`.

**Example:** `{"IsGraphQL":true,"OperationType":1,"RootFields":["hello"],"HasErrors":false}`.

## MCP Fields

Only populated for requests to an [MCP](/ai-management/mcp-gateway/overview) Proxy; absent or empty for other requests.

### MCPStats

Details about the MCP request: `is_mcp`, `jsonrpc_method`, `primitive_type` (`tool`, `resource`, or `prompt`), and `primitive_name`.

**Example:** `{"is_mcp":true,"jsonrpc_method":"tools/call","primitive_type":"tool","primitive_name":"get_current_weather"}`.
