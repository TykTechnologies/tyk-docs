## Prerequisites

- [Kubernetes](https://kubernetes.io/docs/setup/) 1.33.x, 1.34.x, 1.35.x
- [Helm 3+](https://helm.sh/docs/intro/install/)
- Tyk Cloud account with an active Control Plane.

## Instructions

### Step 1: Clone and configure

1. Clone the [tyk-install](https://github.com/TykTechnologies/tyk-install) repository and navigate to the Kubernetes hybrid directory:

    ```bash
    git clone https://github.com/TykTechnologies/tyk-install
    cd tyk-install/kubernetes/helm-hybrid
    ```

2. Copy the example environment file and add your Tyk Cloud credentials:

    ```bash
    cp .env.example .env
    ```

3. Open `.env` and set your credentials (RPC Key, Org ID, etc.).

4. Load environment variables:

    ```bash
    source .env
    ```

### Step 2: Create Namespace and Secrets

```bash
kubectl create namespace tyk-dp

kubectl create secret generic tyk-data-plane-conf \
  --namespace tyk-dp \
  --from-literal=APISecret=$TYK_API_SECRET \
  --from-literal=orgId=$TYK_ORG_ID \
  --from-literal=userApiKey=$TYK_API_KEY \
  --from-literal=groupID=$TYK_GROUP_ID
```

### Step 3: Install Dependencies (Redis)

```bash
helm repo add bitnami https://charts.bitnami.com/bitnami
helm repo update

helm install tyk-redis bitnami/redis \
  --namespace tyk-dp \
  --set auth.enabled=false \
  --version 27.0.8

# Wait for Redis to be ready
kubectl wait --for=condition=ready pod -l app.kubernetes.io/name=redis -n tyk-dp --timeout=300s
```

### Step 4: Install Tyk Data Plane

```bash
helm repo add tyk-helm https://helm.tyk.io/public/helm/charts/
helm repo update

helm install tyk-dp tyk-helm/tyk-data-plane \
  --namespace tyk-dp \
  --values values.yaml
```

Wait for all pods to be running:

```bash
kubectl get pods -n tyk-dp -w
```

## Configuration

The following environment files control each component's settings:

| File | Purpose |
| ---- | ------- |
| `.env` | Credentials and component versions |
| `values.yaml` | Helm chart configuration |

For production deployments, review the [Planning for Production](/planning-for-production) guide and apply the recommendations noted in the config file comments.

## Cleanup

Remove the Tyk Data Plane and dependencies:

```bash
helm uninstall tyk-dp -n tyk-dp
helm uninstall tyk-redis -n tyk-dp
kubectl delete namespace tyk-dp
```
