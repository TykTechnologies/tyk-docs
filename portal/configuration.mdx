# Tyk Portal Webhooks Configuration

This page details the environment variables available for configuring webhooks in the Tyk Developer Portal.

| Environment Variable | Description |
| :--- | :--- |
| `PORTAL_WEBHOOKS_SECRET` | The global secret key used to sign all outgoing webhooks. |
| `PORTAL_WEBHOOKS_PAUSE_DURATION` | The duration (in seconds) to pause a webhook after a failed delivery attempt before retrying. |
| `PORTAL_WEBHOOKS_CACHE_EXPIRATION` | The expiration time (in seconds) for the webhook cache. |
| `PORTAL_WEBHOOKS_CACHE_CLEANUP_INTERVAL` | The interval (in seconds) at which the webhook cache is cleaned up. |
| `PORTAL_WEBHOOKS_DISPATCH_TIMEOUT` | The timeout (in seconds) for dispatching a single webhook request. |
| `PORTAL_WEBHOOKS_TOTAL_WORKERS` | The number of worker goroutines for processing webhooks concurrently. |
| `PORTAL_WEBHOOKS_PROVIDER` | The storage provider for webhooks (e.g., `db`). |
| `PORTAL_WEBHOOKS_DISABLE` | A boolean flag (`true`/`false`) to disable webhooks entirely. |
