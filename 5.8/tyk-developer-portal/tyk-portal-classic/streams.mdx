As of Tyk v5.7.0, you can now publish Tyk Streams APIs to the Tyk Developer Portal.

## How To Set Up

Simply create a [Tyk Streams API](/5.8/api-management/event-driven-apis#getting-started), create a [Policy](/5.8/api-management/gateway-config-managing-classic#create-a-security-policy-with-the-dashboard) which protects it, and then [publish it to the Developer Portal Catalog](/5.8/portal/overview/getting-started#publish-an-api-product).
